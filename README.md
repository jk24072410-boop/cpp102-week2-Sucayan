## Reflection
In this activity, I learned how algorithms, pseudocode, and flowcharts serve as essential tools for
developing program logic. Algorithms helped me break problems into clear step-by-step
instructions, while pseudocode allowed me to express these steps in simple language. 
Flowcharts provided a visual representation of the process, making it easier to understand the flow of 
decisions and operations. Overall, this activity showed me that careful planning through these tools 
leads to more accurate and efficient programming.
